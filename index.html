⚡ How Live Data Works — & How to Upgrade This Dashboard

Right now this is a static HTML file — all historical data is hardcoded. The news feed at the top is already live and pulls real headlines from the GDELT Project API (a free, real-time global news tracker). Here's how the different tiers of "live" work:

🟢 Already Live (This File) The GDELT news feed fetches real headlines every 5 minutes. No setup needed — just open this file in a browser. Uses: api.gdeltproject.org
🟡 Semi-Live (Easy Upgrade) Host this file on any website (Netlify, GitHub Pages, Vercel — all free). A JavaScript setInterval can auto-refresh the full page on a schedule. Add a config.js file you update manually each day with new scores.
🔵 Fully Live (Requires Backend) Connect to NewsAPI, Bing News API, or scrape primary sources. Write a small backend (Node.js or Python, free on Railway/Render) that updates a JSON file daily. This dashboard reads from that JSON file automatically.

The simplest path to fully live data:
  • Create a free account at newsapi.org (100 req/day free)
  • Replace the GDELT fetchNews() call with NewsAPI — it has better Middle East coverage
  • To auto-update the probability/score numbers, either update this file manually each day, or connect to a small Google Sheet via its JSON API (free, no backend needed)
  • For the most powerful setup: Perplexity API can summarise daily events and output structured JSON directly into this dashboard

Abu Dhabi Attack Probability — The daily percentage is the estimated probability that Abu Dhabi (including Al Dhafra Air Base, Zayed International Airport, Ruwais, Habshan, ADNOC infrastructure, and civilian areas) would experience at least one confirmed kinetic strike on that day — whether intercepted debris or a direct hit. It does not measure severity, only occurrence likelihood.

Iran–US War Intensity Score — Represents conflict intensity on a 1–100 scale, calibrated by active strike intensity, diplomatic openings vs. closures, leadership statements, proxy front activity, economic pressure indicators, and third-party mediation status. 100 = full war, no peace path. 1 = imminent ceasefire.
Factor 1 — War Phase
Sets the base rate; escalation/plateau/wind-down phase multiplier applied first
Factor 2 — Strike Targeting
IRGC operational statements explicitly naming targets. Al Dhafra = maximum uplift
Factor 3 — Intercept Rate vs. Debris
Even 90%+ intercepts cause debris casualties. Score includes debris-strike probability
Factor 4 — Diplomatic Signals
Pezeshkian apology, Islamabad talks, Trump pause all reduce daily probability
Factor 5 — Missile Depletion
Iranian stockpile degradation lowers per-day fire volume and probability
Factor 6 — Economic Pressure
Oil above $110 = IRGC targeting energy. Below $100 = de-escalation pressure
